این پروژه برای استفاده آزاد در دسترس قرار گرفته است در جهت کمک به فرهنگ و ادب پارسی زبانان

نکته 1 : 
فونت های استفاده شده در پروژه : 
وزیرمتن : کاملا رایگان 
گلپایگان : فونت شامل لایسنس است.
نسخه فونت گلپایگان موجود در پروژه خریداری شده اما برای استفاده عمومی منع دارد.
لطفا در صورت تمایل به استفاده از فونت گلپایگان حتما لایسنس آن را خریداری کنید
